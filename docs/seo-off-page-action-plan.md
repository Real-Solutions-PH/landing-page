# SEO Off-Page Action Plan — realsolutionsph.com

> **Why this doc exists.** A June 2026 audit flagged this site as "invisible," but on-page,
> technical, and schema/GEO SEO are **already implemented and live** (optimized titles, unique
> meta descriptions, full JSON-LD: Organization/LocalBusiness/WebSite/Breadcrumb/FAQ/BlogPosting,
> canonical 308 apex→www, security headers, dynamic sitemap). The real growth levers now are
> **off-page authority** and **content cadence** — almost none of which is code. This is that work.
>
> The low Bing numbers (≈19 impressions / 3 months) are normal for a **young domain with zero
> backlinks**, not a sign of broken SEO. Authority + content + time is the fix.

## Phase 0 — Foundation & measurement (this week)

- [ ] **Fix Google Search Console access.** The audit was run from an account without GSC access.
      Log in with the Google account that owns the property (or have the owner add your account as
      a user) at search.google.com/search-console. Without GSC you're blind to Google's
      impressions, positions, Core Web Vitals, and indexing errors. **Highest-priority item.**
  - [ ] Verify the `https://www.realsolutionsph.com/` property (DNS or HTML-tag verification).
  - [ ] Submit `https://www.realsolutionsph.com/sitemap.xml`.
  - [ ] Request indexing for the homepage, `/services`, and each `/services/[slug]` + blog post.
- [ ] **Bing Webmaster Tools — run "Site Scan"** (never been run per the audit). It's free and
      surfaces any remaining technical issues across all pages.
- [ ] **Enable IndexNow** in Bing Webmaster Tools so new/updated pages are pushed to Bing instantly.
- [ ] **Lock down NAP (Name, Address, Phone).** The `LocalBusiness` schema currently has no phone.
      Decide on a single public business phone + a consistent address string and use the **exact
      same** NAP everywhere (site, directories, Google Business Profile). Inconsistent NAP is the
      #1 local-SEO killer.

## Phase 1 — Google Business Profile + core authority directories (week 1–2)

> These are the listings most cited by both search engines and AI answer engines (Clutch and
> GoodFirms in particular are heavily referenced by ChatGPT/Perplexity for "best dev company" queries).

- [ ] **Google Business Profile** (business.google.com) — even as a service-area / remote-first
      business based in Batangas. Pick the service-area option if you don't take walk-ins. This is
      the single biggest local-visibility lever and feeds Google's map/AI results.
- [ ] **Clutch.co** — create a profile, add services, request 2–3 client reviews (Clutch verifies
      them by phone/email; verified reviews are what give the profile authority).
- [ ] **GoodFirms** — same playbook as Clutch.
- [ ] **DesignRush**, **UpCity**, **The Manifest**, **TechBehemoths**, **G2** (for any productized
      offering) — create profiles where relevant. Each is a real backlink + a discovery surface.
- [ ] **LinkedIn Company Page** — confirm it matches the `sameAs` in schema
      (`linkedin.com/company/real-solutions-ph`). Post the new blog articles from it.
- [ ] **GitHub org** — confirm `github.com/Real-Solutions-PH` is public, has a pinned profile
      README linking back to the site, and (ideally) 1–2 public repos/tools. Devs and AI crawlers
      both weight this.

## Phase 2 — Philippine local citations & communities (week 2–4)

- [ ] List on PH business directories: **DTI/SEC-adjacent listings**, your local **Batangas
      chamber of commerce**, **Yellow Pages PH**, **PhilippineCompanies**, **BizDirectory PH**.
- [ ] Get listed/active in PH startup & tech communities for mentions and links:
      **StartupPH**, local founder Slack/Discord/FB groups, **DICT**-adjacent SME programs.
- [ ] If you've done university or local-business projects, request a backlink/case-study mention
      from those partners (the highest-trust link type you can get cheaply).

## Phase 3 — Content-driven links & GEO reinforcement (ongoing)

> AI engines already cited the domain ~10× in 3 months with zero effort — content quality is there.
> The job now is to give them more specific, quotable, well-structured pages to cite.

- [ ] **Keep publishing** — see the content cadence below. Each post is a new long-tail entry point
      and a linkable asset.
- [ ] **Repurpose every post** into a LinkedIn post + a short X thread linking back. Cheap, compounding.
- [ ] **Guest posts / mentions**: pitch 1 article/quarter to PH business or tech publications
      (e.g. local startup blogs, BPO/tech outlets). A single editorial backlink outweighs dozens
      of directory links.
- [ ] **HARO-style sourcing / expert quotes**: respond to journalist requests about PH software,
      outsourcing, and SME tech — earns high-authority editorial links.
- [ ] **Get listed in "best software companies in the Philippines" roundups** — reach out to the
      authors of existing listicles; these are exactly what AI engines summarise.

## Content cadence (the one valid audit finding)

The audit's only accurate complaint: **content volume**. The blog system is fully built and
SEO-ready (each post auto-generates BlogPosting + FAQ + Breadcrumb schema and enters the sitemap).
Two starter posts existed; **three more were added 2026-06-09** (system integration, business
process automation, AI for business). Target going forward:

- [ ] **2–4 posts / month**, each targeting one specific long-tail, buyer-intent query.
- [ ] Prioritise clusters already showing demand in Bing: **pricing/cost**, **system integration**,
      **outsourcing/offshore**, **hiring developers PH**.
- [ ] Backlog ideas: "Hiring software developers in the Philippines (rates & models)",
      "Build vs buy for Philippine SMEs", "How to choose a software development partner",
      "Xero/QuickBooks integration guide", "Shopify + inventory integration for PH retailers",
      "ERP alternatives for Philippine SMEs".

## What NOT to do

- ❌ Don't buy bulk backlinks / PBN links — they're the fastest way to a penalty for a young domain.
- ❌ Don't chase volume over relevance — 10 relevant PH/tech links beat 1,000 generic ones.
- ❌ Don't re-implement the June 2026 audit's on-page recommendations — they describe an old version
      of the site and are already done. Re-running the audit against the **current live site** is
      step zero before acting on any future audit.

### Priority order if you only do five things

1. Fix GSC access + submit sitemap.
2. Create Google Business Profile.
3. Get on Clutch + GoodFirms with 2–3 verified reviews.
4. Set a consistent NAP (add the business phone to the site/schema).
5. Keep shipping 2–4 blog posts/month.
